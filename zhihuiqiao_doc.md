# 智汇桥——AI驱动的产学研用一体化智慧协同系统

## 一、项目概述

### 项目名称
智汇桥——AI驱动的产学研用一体化智慧协同系统

### 项目目标
针对当前高校科研合作中存在的信息不对称、资源闲置浪费、学生实践机会少、校企对接难等问题，设计并实现一个连接教师、学生、实验室与校企资源的智慧协同平台。系统通过科研项目智能撮合、校园闲置资源流转、教学辅助个性化支持三大核心模块，为师-生-机协同提供一站式解决方案，提升科研合作效率、资源利用率与学生学习效果。

### 项目定位
本项目以“智能撮合 + 资源共享 + 个性化学习辅助”为核心，融合 Vue 前端、Spring Boot 后端、MyBatis-Plus 数据访问、大模型智能体接入等技术，构建一个面向高校师生与企业合作方的轻量化智慧校园协同平台。系统重点通过后端业务系统与智能体能力的集成，实现“业务系统负责数据管理，智能体负责学习辅助与推荐”的解耦设计，适合作为毕业设计、课程实训和项目答辩展示系统。

## 二、项目主要功能

### 1. 用户与权限管理
支持学生、教师、企业、管理员等多种角色登录，完成用户信息维护、角色权限控制、账号状态管理等基础功能。

### 2. 科研项目智能撮合
- 教师/企业发布科研项目或技术需求
- 学生基于科研画像匹配适合的项目
- 支持项目申请、审核、成员管理全流程
- 提供企业需求发布与检索功能

### 3. 校园闲置资源流转
- 师生发布实验设备、图书资料、办公用品等闲置资源
- 支持资源预约、借用审批、归还记录
- 自动生成资源流转日志

### 4. 教学辅助个性化支持
- 课程资源管理（课程、章节、知识点、学习资料）
- 智能学习问答
- 知识点讲解生成
- 个性化学习计划
- 练习题与错题反馈
- 学习画像与效果评估

### 5. 数据统计看板
通过图表展示科研项目数量、资源流转情况、用户活跃度、学习进度等核心数据，为管理员和教师提供决策支持。

## 三、技术路线

### 前端技术
Vue 3.4、Vite 5、Element Plus 2.7、Pinia、Vue Router、Axios、ECharts

### 后端技术
Spring Boot 3.1.6、JDK 17、MyBatis-Plus 3.5.6、Spring Security、JWT、Maven

### 数据库与存储
MySQL 8.0、Redis 7、MinIO（预留）

### AI 智能体接入
扣子智能体平台、大模型 API、知识库、RAG 检索增强

### 工程化工具
Maven、Git、IDEA、Postman

## 六、实施规划

### 第一阶段：需求分析与方案设计
1. 明确系统用户角色，包括学生、教师、企业、管理员；
2. 梳理核心功能，包括科研撮合、资源流转、教学辅助；

4. 确定智能体接入方式和技术方案。

### 第二阶段：基础环境搭建

3. 初始化 MySQL 数据库，导入建表脚本和基础测试数据；
4. 配置 MyBatis-Plus、Redis、文件存储等基础环境。

### 第三阶段：核心功能开发
1. 完成用户管理、科研项目管理、资源管理、学习资源管理等基础接口；

4. 开发前端主要页面，实现科研撮合、资源流转、学习辅助等功能；
5. 完成前后端接口联调。

### 第四阶段：系统联调与优化
1. 对系统进行完整流程测试，包括登录、项目申请、资源预约、智能问答；

4. 编写测试报告、接口说明和系统设计文档。

### 第五阶段：总结与答辩准备

## 九、项目特色

1. **产学研用一体化**：打通科研项目、资源流转、教学辅助三大场景，形成完整闭环；
2. **智能体辅助学习**：通过智能问答、知识点讲解和学习计划生成，提升学生自主学习效率；
3. **个性化学习路径**：根据学生目标、学习记录和薄弱知识点，生成针对性学习计划；
4. **资源高效流转**：盘活校园闲置资源，提高设备、图书等资产利用率；
5. **业务与 AI 解耦**：后端系统负责用户、项目、资源等业务数据，智能体负责问答和生成能力，结构清晰；
6. **适合毕设展示**：系统功能完整，覆盖数据库、后端、前端、智能体接入、数据可视化等多个技术点。
