### 二、今日工作内容

#### 2.1 科研撮合模块后端开发

完成科研撮合模块的核心后端代码：

- **实体类**：`ResearcherProfile`、`ResearchProject`、`ProjectApplication`、`EnterpriseDemand`
- **Mapper 层**：继承 MyBatis-Plus BaseMapper，实现基础 CRUD
- **Service 层**：
  - `ResearcherProfileService`：科研画像创建、更新、查询
  - `ResearchProjectService`：项目发布、分页列表、详情查询、状态更新
  - `ProjectApplicationService`：提交申请、列表查询、审核通过/拒绝
  - `EnterpriseDemandService`：企业需求发布、分页列表、详情查询
- **Controller 层**：暴露科研撮合模块 RESTful API
- **全文检索**：利用 MySQL ngram 解析器实现项目标题与描述的中文全文搜索

#### 2.2 项目申请与审核接口联调

完成项目申请全流程接口：

- 学生提交项目加入申请
- 教师/项目负责人查看申请列表
- 教师审核申请（通过/拒绝）
- 审核后项目成员数自动更新
- 防止重复申请校验

#### 2.3 资源流转模块后端开发

完成资源流转模块的后端代码：

- **实体类**：`IdleResource`、`ResourceBooking`、`ResourceTransferLog`
- **Mapper 层**：基础 CRUD 与自定义查询
- **Service 层**：
  - `IdleResourceService`：闲置资源发布、分页列表、详情查询、状态更新
  - `ResourceBookingService`：预约提交、列表查询、审批、归还
  - `ResourceTransferLogService`：流转记录自动记录
- **Controller 层**：暴露资源流转模块 RESTful API
- **时间冲突校验**：同一资源在同一时间段内避免重复预约

#### 2.4 前端资源列表页与详情预约页开发

完成前端资源流转相关页面：

- **资源列表页**：
  - 分类筛选（实验设备/图书资料/办公用品/电子数码/场地空间/其他）
  - 状态筛选（可借用/已借出/不可用）
  - 关键词搜索
  - 资源卡片列表（名称/类型/位置/价格/状态）
  - 分页组件
- **资源详情页**：
  - 资源完整信息展示
  - 图片预览
  - 借用规则说明
  - 预约时间段选择
  - 预约申请弹窗（用途说明）
  - 资源所有者审批入口

#### 2.5 模拟数据编写

为科研撮合模块和资源流转模块补充模拟数据：

- 科研项目数据：5 条不同领域的科研项目
- 企业需求数据：3 条企业技术需求
- 闲置资源数据：5 条不同类型的闲置资源
- 科研画像数据：2 条教师科研画像
