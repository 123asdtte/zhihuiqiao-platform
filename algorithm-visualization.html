Kahn 算法会先找到所有入度为 0 的节点（没有前置知识的知识点），逐步向外扩展，最终得到拓扑有序的学习序列。

📝 Kahn 算法伪代码

function kahnTopologicalSort(graph):
  queue ← empty
  sorted ← empty list
 
  // Step 1: 入度为0的节点入队
  for each node in graph:
    if node.inDegree == 0:
      queue.enqueue(node)
 
  // Step 2: BFS 拓扑排序
  while queue is not empty:
    current ← queue.dequeue()
    sorted.append(current)
 
    for each neighbor of current:
      neighbor.inDegree -= 1
      if neighbor.inDegree == 0:
        queue.enqueue(neighbor)
 
  // Step 3: 检测环
  if sorted.size < graph.nodeCount:
    return "存在环！"
  return sorted
